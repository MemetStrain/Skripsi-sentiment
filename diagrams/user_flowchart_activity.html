📊 Tahap 1: Authentication

ENTRY POINT

  • Register: User baru membuat akun
  • Login: User existing login dengan credential
  • Validation: Sistem validasi ke database
  • Auto-Login: Setelah register, langsung login

📊 Tahap 2: Dashboard

MAIN HUB

  • Display: Grafik HMM & harga historis
  • Navigation: Menu untuk pilihan fitur
  • User Check: Role berbeda (User vs Admin)
  • Hub: Semua navigasi melalui sini

📊 Tahap 3: User Paths

FEATURES

  • Prediksi: View chart + input range + predict
  • Berita: List berita + detail + sentimen
  • About: Info sistem & skripsi
  • Back: Semua path kembali ke dashboard

📊 Tahap 4: Admin Path

EXCLUSIVE

  • Update Data: Input harga baru (OHLC)
  • Validation: Format check sebelum submit
  • HMM Update: Recalculate hidden states
  • Dashboard: View system statistics

📊 Tahap 5: Prediction Inference

ML ENGINE

  • Load Model: Load pre-trained model
  • Inference: Kalkulasi prediksi
  • Metrics: Hitung MAPE, R², MAE
  • Visualize: Tampil grafik hasil

📊 Tahap 6: Logout

EXIT POINT

  • Logout: User keluar dari sistem
  • Session: Session dihapus dari server
  • Redirect: Kembali ke halaman login
  • End: Alur berakhir

🔄 Flow Characteristics

  • Sequential: Sebagian besar flow berjalan berurutan
  • Conditional: Ada decision points untuk branching
  • Looping: Retry untuk error validation
  • Hierarchical: Admin features terpisah dari user features
  • Return Paths: Semua path kembali ke dashboard atau menu utama
  • Role-Based: Admin hanya bisa akses Update Data & Admin Dashboard
