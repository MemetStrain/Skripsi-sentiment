Pendahuluan

Proses P5 — Dashboard & Live Inference dimulai dari python manage.py runserver yang menyalakan Django development server di 127.0.0.1:8000. Setiap HTTP request akan di-dispatch oleh URL router (config/urls.py → web/urls.py) ke salah satu dari empat view: dashboard, news, about, atau forecasts_api. Tiga view pertama bersifat sinkron read-and-render; endpoint forecasts_api menjalankan inferensi XGBoost live per request — inilah sumber kompleksitas kontrol terbesar (loop horizon × loop anchor + lru_cache hit/miss + tiga jalur error).

Flowchart ini menampilkan empat sub-alur view, masing-masing diakhiri respons HTTP. Sub-alur forecasts_api diberi detail penuh: validasi parameter (400), dua jalur error fallback (503 untuk artefak hilang, 500 untuk exception lain), dan dua loop bersarang (horizon × anchor) dengan keputusan cache hit/miss serta skip-NaN.

Catatan Implementasi

  • Tanpa autentikasi: seluruh endpoint publik dan read-only. Tidak ada login, sesi, atau CSRF check untuk GET API.
  • Inferensi live: forecasts_api memuat model.pkl + scaler.pkl per (tag, horizon) via joblib.load, di-cache process-wide dengan @lru_cache(maxsize=64). Maksimal kombinasi yang dibutuhkan: 4 tags × 7 horizon × 1 varian CSA = 28 entry — jauh di bawah batas cache.
  • Winners cache: winners.json di-cache dengan @lru_cache(maxsize=1) — file hanya dibaca sekali per proses gunicorn / Vercel.
  • Convergence trail: setiap horizon h-step-ahead diakhiri pada satu titik masa depan bersama (next_trading_day_after(last_date)), sehingga keseluruhan 7 trail bertemu di titik prediksi yang sama.
  • Composite-index avoidance: news view sengaja meng-stream seluruh dokumen news_articles tanpa filter Firestore — filter dan sort dilakukan di Python — untuk menghindari kewajiban composite index (date + sentiment).
  • Pagination clamp: page_number = max(1, min(page_number, total_pages)) menjamin parameter di luar rentang valid tetap merender halaman batas atas/bawah (tanpa 404).
  • Cross-reference: dekomposisi modul (caller→callee) lihat Structure Chart P5.
