About This Project

An automated system for monitoring and forecasting Crude Palm Oil (CPO) prices, built as a thesis project at Universitas Bina Nusantara.

Price Monitoring

Tracks daily CPO prices from global commodity markets and displays trends over time with interactive charts.

Market State Detection

Identifies whether the market is in a Bullish, Bearish, or Neutral state using statistical models trained on historical price data.

Price Forecasting

Forecasts CPO prices 1–7 days ahead using multiple machine learning models that combine price history with news sentiment signals.

Data Sources

  • 1
    CPO Price Data — Daily commodity prices fetched automatically from Investing.com, covering open, high, low, and close values.
  • 2
    MPOB News — Articles from the Malaysian Palm Oil Board, automatically scraped daily and analyzed for market sentiment.

Fully Automated Pipeline

No manual data uploads are needed. Every day, a scheduled cloud job:

  • Fetches the latest CPO closing price
  • Scrapes new MPOB news articles
  • Runs sentiment analysis on new articles
  • Updates market state predictions
  • Refreshes all price forecasts
